PackRight is een B2B-verpakkingsbedrijf gevestigd in Rijsenhout, Nederland. Zij zijn één van de beste in de packaging business en onderhouden vele bekende klanten zoals RITUALS, LAGEEN, KEUNE.

PackRight biedt verpakkingen voor merken zoals Rituals, Lageen, Keune, Blond Amsterdam, etc. Ze moesten een website bouwen die hun hoogwaardige verpakkingsmaterialen op één plek tentoonstelt.

1. Van hoge kwaliteit
2. Volledig op maat gemaakt

 

 

Bezoek Website

PHP
CodeIgniter Framework
MySQL
HTML & CSS

De primaire focus was om hun product op een gebruiksvriendelijke en visueel aantrekkelijke manier te kunnen presenteren.

De belangrijkste kenmerken van de website zijn de volgende -

a) PHP ontwikkelde Backend met CMS en File Manager

b) Mogelijkheid om producten te beheren / bewerken / toevoegen / verwijderen

c) Mogelijkheid om productcategorieën en subcategorieën te maken

h) Volledig aangepast ontwerp zonder een template te gebruiken

i) 3 soorten aangepaste productweergave

- Zoom in & Zoom out

- Grote maat

- Draaibare productweergave van 360 graden

Tijdlijn overeengekomen om het project te voltooien was 7 weken.
